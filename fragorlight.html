Svar på frågor
Vad betyder HTTP och vad är ett kommunikationsprotokoll?

HTTP betyder Hypertext Transfer Protocol. Kommunikationsprotokoll, när det gäller HTTP, är överförning av websidor över internet.

Vad betyder FTP?

FTP står för File Transfer Protocol, och används för att överföra text och binära datafiler.

Vad kallas HTTPS på fackspråk?

HTTPS står för Hypertext Transfer Protocol Secure. HTTPS är ett protokoll för krypterad transport av HTTP-protokollet.

Vad kallas "portal.boden.se" på fackspråk

"portal" kallas för ett subdomain, den visar vilken del av sida du befinner dig på.

"boden" kallas för ett secondary-level domain, den visar namnet på hemsidan.

".se" kallas för ett top-level domain, detta specificerar vilket enhet den är registrerad som på internet. ".se" specificerar också att hemsidan är svensk.

Vad kallas "/db/web/external.ns" på fackspråk?

"/db/web/external.ns" kallas för ett subdirectory. De specificerar vilken del av hemsida du är på.

Vad betyder HTML och vad innebär det?

HTML betyder HyperText Markup Language och används för att göra hemsidor (som denna).

Vad betyder CSS och hur tillämpar man det på hemsidor?

CSS betyder Cascading Style Sheets och är en stilmall för hemsidor.
